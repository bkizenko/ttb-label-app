# Demo / test images

All demo and test images live under this folder: **`public/demo/`** (about 80+ image files total).

**Folders:**
- **wine images/** — DAOU wine (JPG, HEIC)
- **stone's throw/** — Stone's Throw IPA (PNG, JPG)
- **vodka images/** — Absolut vodka (HEIC, JPG)
- **old tom distillery test images/** — Old Tom bourbon (PNG, JPG)

Demo presets use a subset of these; paths are in `src/data/presets.ts`. To see every file, open `public/demo/` in your editor or run:  
`find public/demo -type f \( -name "*.jpg" -o -name "*.png" -o -name "*.HEIC" \)`
